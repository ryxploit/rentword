WordPress theme para rentas vacacionales con búsqueda avanzada e integración API.

== Description ==

RentWord Theme para negocios de rentas vacacionales y agencias inmobiliarias.

= Características =

* API Integration
* Advanced Search
* Responsive Design
* Theme Options Panel
* Shortcodes y Widgets
* Gutenberg Blocks

= Templates =

* Home Page
* Properties Listing
* Single Property

= Shortcodes =

* [rentword_properties]
* [rentword_search]
* [rentword_featured_slider]
* [rentword_map]

== Installation ==

1. Upload theme ZIP
2. Go to Appearance > Themes
3. Activate RentWord Theme
4. Configure API in RentWord > API Settings
